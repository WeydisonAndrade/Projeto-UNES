# Projeto-UNES
## Um simples projeto de uma página de universidade
Este projeto é parte prática do curso de Desenvolvimento Web com Python e Django, apenas para finalidade de colocar em prática os conhecimentos adquiridos até a módulo referente ao projeto, foi usado HTML, CSS inline

Obs: Este projeto foi apenas para prática dos conhecimentos adquiridos até o módulo, não foram levados em conta algumas questões de boas práticas como não usar tabelas para formatar a página, não fazer quebrar de linha com br em label e inputs e responsividade...conteúdo esse que vai ser estudado posteriormente no decorrer do curso além de Boostrap, design responsivo, Divs, Django e muito mais...

## Nesta atividade prática fiz uso de tags de:

*Parágrafos

*Lista ordenadas e não ordenadas

*Imagens

*Links

*Tabelas

*Formulários
